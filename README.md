# Hugo theme

## Installation 
Add the repository into your Hugo Project repository as a submodule:
```
git submodule add git@github.com:sebousan/hugo-theme themes/hugo-theme
```
## Fonctionnalités
Types supportés par le thème
* posts
* authors (taxonomy)
* categories (taxonomy)
* projects
* types (taxonomy)
* case studies
* publications
* pages

## Framework
