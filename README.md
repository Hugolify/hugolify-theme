# Hugo theme

## Install
Add the repository into your Hugo Project repository as a submodule:
```
git submodule add git@github.com:sebousan/hugolify-theme themes/hugolify-theme
```

## Features
Hugo sections available in theme and admin

* pages
* posts
* authors (taxonomy)
* categories (taxonomy)
* tags (taxonomy)
* projects
* types (taxonomy)
* case studies
* publications
